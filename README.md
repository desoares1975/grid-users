# Aplicação 1 - Grid usuários

## Getting started:

```bash
$ git clone git@github.com:desoares1975/grid-users.git
```
### After clonning on the same folder:

```bash
$ cd grid-users
```
```bash
$ npm install
```
## Starting application

```bash
$ node index.js
```
